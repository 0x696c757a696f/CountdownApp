# 随机专注计时器 3.0 发布说明

发布日期：2026-07-21  
适用平台：Windows 10/11 x64  
发布形态：免安装单文件（PyInstaller onefile）

## 版本概览

随机专注计时器 3.0 是一次以可靠性、提醒体验和界面可用性为重点的更新。它保留 Classic 随机提醒，并加入“前密、中疏、后密”的 V2 三阶段算法，以及可选的注意力反馈和持续背景音。

## 主要功能

- 使用 `secrets.SystemRandom` 从操作系统安全随机源抽取提醒间隔；
- Classic 统一随机区间与 V2 三阶段随机算法；
- 可选“仍在任务”“刚才走神”“正在心流，延后下次提醒”反馈；
- 根据当前周期反馈自动缩短或延长下一次提醒间隔；
- 独立的微休息开始铃和回归专注铃；
- 白／粉红／棕／灰基础噪音、内置风雨雷暴／雨声录音与九种 Solfeggio 频率，可分三层自由组合持续播放；
- 平衡提醒与强干预提醒；
- 暂停、停止、系统托盘、开机可见启动和开机静默启动；
- 可选悬浮计时条与 Windows 全局快捷键；
- 专注结束后由用户决定是否开始大休息。

## 可靠性改进

- 使用单调时钟统一专注、暂停、微休息和大休息计时，避免 GUI 延迟造成漂移；
- 增加完整输入校验和“本轮可能没有提醒”的启动前提示；
- 暂停时正确关闭可见提醒，不误播回归铃；
- 音频设备异常时自动重试，并在界面显示背景音播放失败状态；
- 单实例运行，避免重复启动两套计时器；
- 自动清理指向旧安装目录的开机启动项；
- 配置 schema v1 可迁移至 v2，并保留原有用户偏好；
- 配置损坏时保留原文件，再使用默认设置启动；
- 安装目录不可写时仍可使用内存默认设置，并明确提示设置无法保存；
- 强提醒覆盖完整的多显示器虚拟桌面，并保留 Esc 退出入口；
- 启动异常、版本环境和提醒触发信息会写入滚动日志。
- 排除托盘图标不使用的 Pillow 可选解码器；加入两段环境录音和多尺寸图标后，单文件约为 26.8 MB；
- 背景音 PCM 在工作线程生成，真实环境录音使用独立循环声道；快速切换只应用最后一次选择，停止或退出不会让过期任务恢复播放；
- Windows CI 增加真实 onefile 构建和空目录首次启动冒烟验证；

## UI/UX 改进

- 重新整理主设置页、更多设置和 V2 设置窗口；
- 产品标题、托盘提示和窗口标题统一使用中文；算法与频率保留 `Classic`、`V2`、`Solfeggio` 和 `Hz` 标准名称；
- 设置页支持滚动条和鼠标滚轮，适配较小屏幕与高 DPI；
- 设置页滚动条随 DPI 保持易操作宽度，短数值输入框不再横向铺满卡片；
- Windows 启用第二版每显示器 DPI 感知，并按显示器缩放分别加载标题栏和任务栏图标；
- 主界面、运行界面、微休息界面和 V2 设置窗口按当前 DPI 保持一致的逻辑尺寸；
- 运行界面采用紧凑布局，减少无意义留白；
- 重新设计应用图标，并统一 EXE、Tk 窗口与系统托盘的图标资源；
- 运行期间可展开背景音控制条并即时调整，不会重置专注计时；
- 操作按钮按用途对齐分组，降低误操作概率；
- 自适应反馈选项增加作用说明，明确反馈只影响下一次提醒。
- 主设置和提醒设置补充全屏触发规则，明确 Classic、V2 各阶段及两种提醒预设的表现；
- 修复高 DPI 下运行页底部操作按钮和悬浮计时阶段文字被裁切的问题；阶段说明变化时会重新测量布局，普通计时刷新不会重复测量；
- 悬浮计时被右键或 × 隐藏后，可在专注运行或暂停期间通过托盘恢复；非专注状态不显示该托盘入口。
- 提醒小窗改为内容驱动布局，中文按钮按实际文字宽度显示，并在小屏幕上自动换行；
- 主窗口、提醒小窗、全屏提醒、悬浮计时和 V2 设置窗显式复用同一套标题栏与任务栏图标，不再依赖 Tk 子窗口继承。
- 修复提醒窗在隐藏状态下预先设置置顶后，Windows 偶尔不将其显示到前台的问题；窗口现在会先完成映射，再重新置顶并抬升。
- 悬浮计时改为 Windows 工具窗口，不再单独占用任务栏或 Alt+Tab；仍可通过系统托盘恢复显示。

## 运行方式

本版本不要求用户另外安装 Python、pygame、Tkinter 或其他 Python 包。

1. 将 `CountdownApp.exe` 放在普通用户可写目录；
2. 双击 `CountdownApp.exe` 启动。

Python、Tk、pygame、托盘图标和内置音频均封装在 `CountdownApp.exe` 中。程序启动时会将运行组件解压至系统临时目录，因此首次启动可能比目录版稍慢。

首次运行会在 EXE 同目录自动生成用户配置与日志：

- `settings.json`
- `Logs\countdown.log`

因此程序目录需要具有写入权限。建议放在用户文档、桌面或其他普通用户可写目录，不要直接放入受保护的 `Program Files` 目录。

## 已知限制

- 当前个人测试构建没有数字签名，Windows 可能显示“未知发布者”；
- PyInstaller 单文件每次启动都需要解压运行组件；当前机器实测通常约 3–5 秒，具体时间受磁盘和安全软件扫描影响；
- 尚未在全新的 Windows 虚拟机中完成独立冒烟测试；
- 尚未完成无人干预的完整 90 分钟实机周期测试；
- `secrets.SystemRandom` 是操作系统级密码学安全随机，不代表外接硬件产生的物理真随机；
- Solfeggio 频率仅作为声音选项，不作医疗或认知效果承诺。
